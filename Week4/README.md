## Class
* 클래스는 하나의 type 이다 (int, double etc... 처럼)
* 단 Java가 정해놓은 타입이 아니라 사용자가 직접 정의하는 타입이므로 사용자 정의 타입이라고 부르기도 한다.
* 사용법은 다음과 같다

<pre>
<code>
public class Main{
    public static void main(String[] args){
        int number = 0;
        Person person = new Person(); // Person이 이미 정의되어 있다고 가정한다.
    }
}
</code>
</pre>
